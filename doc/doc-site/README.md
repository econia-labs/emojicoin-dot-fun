# emojicoin dot fun Docs

## Building locally

1. Install [Homebrew] or a similar package manager.

1. Install [pnpm]:

   ```zsh
   brew install pnpm
   ```

1. Install the docs site package dependencies:

   ```zsh
   pnpm install
   ```

1. To serve a local site preview:

   ```zsh
   pnpm start
   ```

1. Open <http://localhost:3000>

[homebrew]: https://brew.sh
[pnpm]: https://pnpm.io/
