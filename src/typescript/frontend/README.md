# Emojicoin Frontend

## Getting Started

First, install:

```bash
pnpm i
```

To run the app:

```bash
pnpm run dev
```

## Building

### For production

```bash
pnpm run build
```

### To debug

```bash
pnpm run build:debug
```

This project uses [`next/font`].

[`next/font`]: (https://nextjs.org/docs/basic-features/font-optimization)
